3.리액트 프로젝트 세팅
npm install -g create-react-app // create-react-app -V(오류? powercell에선 오류 cmd에선 작동)


-리액트 프로젝트 폴더 만들기
create-react-app foldername(카멜)혹은 FolderName(파스칼)

-리액트 서버환경 실행
npm run start

.gitignore .txt파일로 생성을 해야 노드모듈즈를 제외하고 깃 파일 관리를 원활히 할 수있다.
